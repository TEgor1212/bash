## Bash-скрипты для учебного задания

В этом репозитории собраны простые Bash-скрипты для отработки базовых навыков работы в терминале и написания shell-программ.

Все примеры рассчитаны на запуск в Linux/macOS или в WSL. Для редактирования кода рекомендуется использовать **VSCode**.

- **1. Приветствие пользователя — `hello_user.sh`**  
  Скрипт запрашивает у пользователя имя и выводит приветствие.  
  Пример запуска:
  ```bash
  bash hello_user.sh
  ```

- **2. Калькулятор суммы — `sum_calculator.sh`**  
  Запрашивает два числа (целые или вещественные) и выводит их сумму. При наличии `bc` умеет считать с дробями.  
  ```bash
  bash sum_calculator.sh
  ```

- **3. Проверка чётности числа — `check_even.sh`**  
  Определяет, является ли введённое целое число чётным или нечётным.  
  ```bash
  bash check_even.sh
  ```

- **4. Создатель структуры веб-проекта — `create_web_project.sh`**  
  Создаёт папку проекта со структурой:
  ```text
  my-project/
  ├── index.html
  ├── css/
  │   └── style.css
  └── js/
      └── script.js
  ```
  При запуске скрипт спрашивает имя проекта и создаёт соответствующую директорию:
  ```bash
  bash create_web_project.sh
  ```

- **5. Счётчик строк в файле — `count_lines.sh`**  
  Запрашивает путь к файлу и выводит количество строк в нём (использует `wc -l`).  
  ```bash
  bash count_lines.sh
  ```

- **6. Генератор паролей — `gen_password.sh`**  
  Генерирует случайный пароль длиной **8 символов** из букв, цифр и нескольких спецсимволов.  
  ```bash
  bash gen_password.sh
  ```

- **7. Поиск файлов по расширению — `find_by_ext.sh`**  
  Ищет файлы с заданным расширением в текущей директории (например, `txt` или `.sh`).  
  ```bash
  bash find_by_ext.sh
  ```

### Как запустить скрипты

- **Сделать скрипты исполняемыми** (один раз):
  ```bash
  chmod +x *.sh
  ```
- **Запускать**:
  ```bash
  ./hello_user.sh
  ./sum_calculator.sh
  # и т.д.
  ```

### Работа с GitHub или Gitflic

- **Создайте новый репозиторий** на GitHub или Gitflic (без автогенерации файлов, либо с уже существующим `LICENSE`/`README.md`).
- В локальной папке (где лежат эти скрипты) выполните:
  ```bash
  git init
  git add .
  git commit -m "Bash scripts homework"
  git branch -M main
  git remote add origin <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
  git push -u origin main
  ```
- Отправьте преподавателю **ссылку на репозиторий** или на директорию со скриптами.
